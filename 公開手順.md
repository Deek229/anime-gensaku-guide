# 無料でインターネット公開する手順（Render）

**所要時間: 約20分**  
**費用: 0円**（Renderの無料プラン）

公開後のURL例: `https://anime-gensaku-guide.onrender.com`

---

## 準備するもの

- GitHubアカウント（無料）→ https://github.com/signup
- Renderアカウント（無料）→ https://render.com （GitHubでログイン可）

---

## 手順1: GitHubにコードを上げる

### A. Gitを入れる（まだの場合）

PowerShellで:

```powershell
winget install Git.Git --accept-package-agreements --accept-source-agreements
```

**PCを一度再起動**するか、新しいPowerShellを開いてください。

### B. リポジトリ作成

1. https://github.com/new を開く
2. Repository name: `anime-gensaku-guide`（任意）
3. **Public** を選択
4. 「Create repository」をクリック

### C. コードをアップロード

GitHubの画面で **「uploading an existing file」** をクリックし、  
`11_名称未定Web作り` フォルダの中身を**すべて**ドラッグ＆ドロップ:

- `app.py`, `config.py`, `requirements.txt`, `render.yaml` など
- `static/` フォルダ
- `templates/` フォルダ
- `data/works.json`（**必須**）
- `tools/` フォルダ

**アップロードしないもの:** `.env`（秘密情報）、`data/cache/`（キャッシュ）

「Commit changes」を押す。

---

## 手順2: Renderで公開

1. https://dashboard.render.com/ を開く
2. **New +** → **Web Service**
3. GitHubの `anime-gensaku-guide` リポジトリを **Connect**
4. 設定:

| 項目 | 値 |
|------|-----|
| Name | `anime-gensaku-guide` |
| Region | Singapore（日本に近い） |
| Branch | main |
| Runtime | **Python 3** |
| Build Command | `pip install -r requirements.txt && python tools/seed_works.py` |
| Start Command | `uvicorn app:app --host 0.0.0.0 --port $PORT` |
| Instance Type | **Free** |

5. **Create Web Service** をクリック
6. 5〜10分待つ → **Live** になったらURLが発行される

---

## 手順3: Amazonアソシエイトに登録

RenderのURL（例 `https://anime-gensaku-guide.onrender.com`）を  
アソシエイト申請の「ウェブサイト」欄に入力。

承認後、Renderの **Environment** に追加:

```
AMAZON_ASSOCIATE_TAG=あなたのタグ-22
```

保存すると自動で再デプロイされます。

---

## 注意（無料プラン）

- 15分アクセスがないとスリープ → 最初のアクセスに30秒ほどかかることがある
- それでもAmazon審査・日常利用には十分

---

## うまくいかないとき

- ビルドログで `pip install` が失敗 → `requirements.txt` がアップロードされているか確認
- ページが真っ白 → `data/works.json` があるか確認
- ローカル確認: `python -m uvicorn app:app --port 8052` → http://127.0.0.1:8052

# インデックス登録チェックリスト

Google スプレッドシートで、Search Console のインデックス登録状況を管理するためのリストです。

## このリストに含まれるページ

| 種類 | 件数 | 例 |
|------|------|-----|
| トップ・ランキング | 2 | `/` `/rankings` |
| 作品詳細ページ | 87 | `/works/無職転生ⅲ-異世界行ったら本気だす` など |

**合計 89 ページ**（`data/works.json` と `app.py` の sitemap と同じ内容）

> シーズン切り替えはトップのプルダウン、または `/?season=2026-spring` / `/?season=2026-summer` で行えます。デフォルトは2026年夏アニメです。

## 列の意味

| 列名 | 使い方 |
|------|--------|
| **優先度** | ★★★＝最優先（人気上位5作品）、★★☆＝次点（次の5作品＋ハブ）、★☆☆＝その他 |
| **ページ名** | ページのタイトル（管理用） |
| **URL（フル）** | Search Console に貼り付ける完全 URL |
| **インデックス登録リクエスト** | GSC で「インデックス登録をリクエスト」したら「済」や日付を記入 |
| **site:検索で確認** | Google で `site:anime-gensaku-guide.onrender.com キーワード` を検索し、表示されたら「済」 |
| **メモ** | 自由記入 |

## Google スプレッドシートへの取り込み方

### 方法A：新しいスプレッドシートに取り込む（おすすめ）

1. [Google スプレッドシート](https://sheets.google.com) を開く
2. 左上 **「空白」** で新しいシートを作成
3. メニュー **「ファイル」→「インポート」**
4. **「アップロード」** タブで `インデックス登録チェックリスト.csv` を選ぶ
5. インポート設定：
   - **場所**：「新しいシートを挿入する」
   - **区切り文字の種類**：「カンマ」
   - **文字コード**：「UTF-8」（文字化けする場合は「Unicode (UTF-8)」を試す）
6. **「データをインポート」** をクリック

### 方法B：既存のシートに追記する

1. スプレッドシートを開く
2. **「ファイル」→「インポート」**
3. CSV をアップロード
4. **場所**：「現在のシートを置換する」または「現在のシートの下に挿入する」

## 使い方の流れ（1ページあたり）

1. **URL（フル）** をコピー
2. [Google Search Console](https://search.google.com/search-console) → **URL 検査** に貼り付け
3. 「インデックス登録をリクエスト」を実行したら、**インデックス登録リクエスト** 列に「済」や日付を書く
4. 数日後、Google 検索で `site:anime-gensaku-guide.onrender.com` とページ名で検索
5. 検索結果に出たら **site:検索で確認** 列に「済」と日付を書く

## 優先度の目安

- **★★★ から順に** 登録リクエストすると効率的です
- トップ・ランキングは 6/28 時点で site: 検索確認済み（CSV に反映済み）

## 作品を追加したとき

`data/works.json` に作品を追加したあと、次のコマンドで CSV を再生成できます。

```
python tools/generate_index_checklist.py
```

生成後、スプレッドシートを再度インポートするか、追加分だけ手でコピーしてください。
